# مشروع إدخال بيانات صقور

مشروع بسيط لإدخال وإدارة بيانات الصقور (API + واجهة إدخال HTML).

متطلبات:
- Node.js >= 16
- Git (اختياري)

التثبيت والتشغيل:
1. انسخ المشروع:
   git clone <repo-url> || انسخ المجلد محلياً

2. ثبّت الحزم:
   npm install

3. إعداد قاعدة البيانات (Prisma + SQLite):
   npx prisma generate
   npx prisma migrate dev --name init

4. شغّل الخادم:
   npm run dev

5. افتح الواجهة:
   افتح المتصفح على: http://localhost:4000/

ملاحظات:
- صور الصقور تُخزّن في مجلد uploads/
- قاعدة البيانات افتراضياً SQLite في ملف `./dev.db`
- النقاط التي يمكن تطويرها لاحقاً: تسجيل مستخدمين/صلاحيات، واجهة React باللغة العربية، رفع عدة صور لكل صقر، تصدير CSV/PDF.
